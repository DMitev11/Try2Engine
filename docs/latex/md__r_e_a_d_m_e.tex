Project with modular architecture.  Every module in the project should be implemented with the intend that its libraries could change in the future.  This project adheres to the principle that every module should be initially implemented as an {\bfseries{adapter}}, {\bfseries{facade}}, {\bfseries{template}} or a similar structural and creational patterns. This approach enables the internals of each module to evolve independently, while the code that utilizes each module remains unchanged.  {\itshape {\itshape Creational}} patterns are up to personal preference.
