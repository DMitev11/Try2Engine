\doxysubsection{Detailed Description}
Separate instances of created input systems with different origin should be allowed. ~\newline
 Emits events on detected implemented input from input devices. 

\begin{DoxyWarning}{Warning}
Contains intances of connected controllers which need to be destroyed, deleted and cleaned up before a {\ttfamily terminate} of each system 
\end{DoxyWarning}
