\doxysubsection{Detailed Description}
Controlling stack for layers. ~\newline
 Calls \doxylink{classutils_1_1_layer_component}{Layer\+Component} events\+: 


\begin{DoxyItemize}
\item On\+Attach, when a layer can been created and attached to the current layer stack
\item On\+Tick, during each tick update of the layer stack
\item On\+Detach, when a layer have been removed from the layer stack
\end{DoxyItemize}

This class follows \doxylink{classutils_1_1_loop_events}{Loop\+Events} modal, present in most game-\/related applications/simulations\+:


\begin{DoxyItemize}
\item init -\/ initialization logic for the current layer stack, if there is initialization logic necessary specifically for this layer stack (initialize libraries, add specific layers etc)
\item tick -\/ on every successful loop of the program
\item shutdown -\/ called once the application is about to end
\end{DoxyItemize}

\begin{DoxyWarning}{Warning}
If project\textquotesingle{}s structure have been changed to include several layer stacks -\/ override \textquotesingle{}terminate\textquotesingle{} as a method to destroy and clear a single layer stack 
\end{DoxyWarning}
